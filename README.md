Evidencia Actividad 3.5 - Resultado de implementación de la API

Comentario de aprendizaje:
En este ejercicio aprendí a crear y configurar una API básica utilizando Python y el framework Flask. Comprendí cómo definir rutas (endpoints) y la diferencia clave entre los métodos HTTP: utilicé el método GET para consultar datos y el método POST para enviar nueva información al servidor. Además, aprendí a utilizar la herramienta Postman para simular peticiones de un cliente (enviando datos en formato JSON en el Body) y verificar que la API responde correctamente con los códigos de estado adecuados, como el 201 Created.
